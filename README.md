# FemBoy-Colizeum-Bot

## Commands:

/help - вывод основных команд

/start - регистрация в систее бота

/create_femboy <name> - создание своего персонажа

/profile - просмотреть профиль персонажа

/shop - магазин

/duel <@username> - вызвать пользователя на дуэль

/train - провести тренировочный бой с персонажем-тренером
